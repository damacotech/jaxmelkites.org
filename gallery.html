Facebook Photo Workflow

A real FTP-friendly gallery pipeline.

Static FTP hosting cannot securely call Facebook directly from a visitor’s browser. Instead, use the included script tools/facebook-gallery-sync.js with a Meta Page access token. It downloads Facebook Page photos into assets/facebook-gallery/ and writes gallery/data/photos.json.

  1. Create or use a Meta app with permission to read the Page photos.
  2. Run the sync script locally with FACEBOOK_PAGE_ID and FACEBOOK_ACCESS_TOKEN.
  3. Upload the updated assets/facebook-gallery/ folder and gallery/data/photos.json by FTP.
